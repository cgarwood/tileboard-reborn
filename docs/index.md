# TileBoard Reborn

TileBoard Reborn is a modern, configurable dashboard for Home Assistant, designed for wall-mounted tablets and kiosk displays.

## Features

- Real-time Home Assistant entity state updates via WebSocket
- Configurable widget grid layout with per-page customization
- Screensaver with weather, calendar, and image slides
- Restrictions and PIN protection for sensitive controls
- Camera widget with snapshot and live stream modes
- Dynamic widget properties driven by entity state (backgrounds, icons, labels)
- Home Assistant addon support for easy installation

## Getting Started

- [Installation](installation.md) — install as an HA addon or run standalone
